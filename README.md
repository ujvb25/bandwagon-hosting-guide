## FAQ：搬瓦工套餐常见问题

**Q：搬瓦工支持支付宝吗？**
A：支持。支付宝、PayPal、UnionPay（银联）、信用卡都可以，国内用户结算不是问题。

**Q：CN2 GIA-E 和香港套餐能迁移到对方的机房吗？**
A：CN2 GIA-E 套餐的可选机房列表里包含部分香港和日本机房，但这取决于当时实际开放情况。Ultra 系列（香港/日本）购买后也可以在各自系列的机房内切换。两个系列是独立产品，不能直接互换。

**Q：KVM 套餐能升级到 CN2 GIA-E 吗？**
A：不能直接升级，是两个不同的产品系列，需要单独购买。

**Q：搬瓦工的限量版套餐什么时候有货？**
A：没有固定规律，不定期上架，卖完为止。官网直接看实时库存是最准确的方式。

**Q：流量用完了怎么办？**
A：流量耗尽后 VPS 会暂停网络访问（Data transfer limit exceeded 状态），等到下个计费周期自动恢复，也可以后台手动申请临时恢复。
